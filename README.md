# Google Data Analytics Capstone Project

Author: Aaron Liu

Tools: SQL (DuckDB/MotherDuck), R (tidyverse, ggplot2, leaflet, ggiraph, R markdown).

Objective: This project analyzes bike rideshare data from October 2024 to October 2025 to expose behavioral differences between members and casual riders and propose strategies for converting casual riders to members.
